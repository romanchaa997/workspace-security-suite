# Code of Conduct

## Our Commitment

We are committed to providing a welcoming and inspiring community for all. We expect all participants to uphold these values in all interactions within our project.

## Expected Behavior

- Be respectful and inclusive of diverse perspectives and backgrounds
- Use welcoming and inclusive language
- Be patient and understanding in technical discussions
- Focus on constructive criticism and improvement
- Report any violations of this code of conduct

## Unacceptable Behavior

- Harassment, discrimination, or intimidation
- Offensive comments related to gender, sexual orientation, disability, or other protected characteristics
- Deliberate intimidation, stalking, or following
- Unwelcome sexual attention or advances
- Any other conduct that creates a hostile environment

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainers at conduct@workspace-security.dev.

## Enforcement

Violations of this code of conduct may result in:
- Warning or temporary suspension from the community
- Removal from the project
- Banning from future participation

## Attribution

This Code of Conduct is adapted from the Contributor Covenant and the Python Community Code of Conduct.
